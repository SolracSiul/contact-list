## Validação da aplicação via POSTMAN:
Disponibilizamos no diretorio do projeto uma coleção do postman para ser utilizada e validar as funcionalidades requisitadas.

`Coleção: postman-helper.json`

### Recuperar o userId

Para recuperar o userId do seu usuario e realize uma chamada para o endpoint `Listar usuarios`

### ATENÇÃO !!

Após realizar a importação da biblioteca no ambiente do postman existe um detalhe importante de atenção. Alguns endpoints precisam de um userId para funcionar corretamente.
Dado isso será necessario que o userId seja preenchido manualmente. Para realizar esse preenchimento siga até a opção `Edit` da biblioteca do Postman e procure a aba `Variable` e atualize o valor da variavel userId.
